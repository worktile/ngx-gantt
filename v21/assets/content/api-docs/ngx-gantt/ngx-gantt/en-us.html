ngx-gantt

Gantt chart main component

Name Type Default Description
items

Task data items array. Set separately for list display; set group_id to associate with groups for grouped display; supports tree structure (children)
groups

Grouped data array. Items need to set group_id to associate, supports expand/collapse
baselineItems

Baseline comparison data, displayed below task bars
viewType

View type: hour, day, week, month, quarter, year, default month
start

Start time (Unix timestamp, seconds). Auto-calculated from items if not set
end

End time (Unix timestamp, seconds). Auto-calculated from items if not set
showTodayLine

Whether to display today line
draggable

Whether to allow dragging task bars to modify time
styles

Style configuration (theme, row height, task bar height, etc.)
showToolbar

Whether to display toolbar
toolbarOptions

Toolbar configuration, can configure list of view types to display
viewOptions

View configuration (time tick format, unit width, scroll loading, etc.)
linkOptions

Link relationship configuration (dependency types, arrows, line types, etc.)
disabledLoadOnScroll

Whether to disable scroll loading, default true
selectable

Whether to enable row selection
multiple

Whether to enable multiple selection mode (requires selectable to be true)
quickTimeFocus

Whether to enable quick time focus (click task bar endpoints to quickly locate)
maxLevel

Maximum display level, default 2
async

Whether to enable asynchronous loading of sub-tasks
childrenResolve

Async loading sub-tasks function, returns Observable<GanttItem[]>
linkable

Whether to allow creating task link relationships
loading

Whether to display loading state
virtualScrollEnabled

Whether to enable virtual scrolling, default true
loadingDelay

Loading delay time (milliseconds), used to avoid flickering, default 0
loadOnScroll

Scroll loading event, event object contains start and end timestamps
dragStarted

Task bar drag start event
dragMoved

Task bar drag move event
dragEnded

Task bar drag end event
barClick

Task bar click event
viewChange

View type change event
expandChange

Expand/collapse state change event
linkDragStarted

Link drag start event
linkDragEnded

Link drag end event
lineClick

Link line click event
selectedChange

Task selection change event
virtualScrolledIndexChange

Virtual scroll index change event
#bar

Task bar custom template (bar type)
#range

Range custom template (range type)
#item

Task item custom template
#baseline

Baseline custom template
#group

Group custom template
#groupHeader

Group header custom template
#toolbar

Toolbar custom template
#footer

Footer custom template
