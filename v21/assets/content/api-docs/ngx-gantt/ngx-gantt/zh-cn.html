ngx-gantt

甘特图主组件

Name Type Default Description
items

任务数据项数组，单独设置以列表展示；设置 group_id 可关联到 groups 进行分组展示；支持树形结构（children）
groups

分组数据数组，items 需设置 group_id 关联，支持展开/收起
baselineItems

基线对比数据，显示在任务条下方
viewType

视图类型：hour、day、week、month、quarter、year，默认 month
start

开始时间（Unix 时间戳，秒）。不设置则根据 items 自动计算
end

结束时间（Unix 时间戳，秒）。不设置则根据 items 自动计算
showTodayLine

是否显示今天线
draggable

是否允许拖拽任务条修改时间
styles

样式配置（主题、行高、任务条高度等）
showToolbar

是否显示工具栏
toolbarOptions

工具栏配置，可配置显示的视图类型列表
viewOptions

视图配置（时间刻度格式、单位宽度、滚动加载等）
linkOptions

关联关系配置（依赖类型、箭头、连线类型等）
disabledLoadOnScroll

是否禁用滚动加载，默认 true
selectable

是否启用行选择功能
multiple

是否启用多选模式（需 selectable 为 true）
quickTimeFocus

是否启用快速时间聚焦（点击任务条端点快速定位）
maxLevel

最大展示层级，默认 2
async

是否启用异步加载子任务
childrenResolve

异步加载子任务函数，返回 Observable<GanttItem[]>
linkable

是否允许创建任务关联关系
loading

是否显示加载状态
virtualScrollEnabled

是否启用虚拟滚动，默认 true
loadingDelay

加载延迟时间（毫秒），用于避免闪烁，默认 0
loadOnScroll

滚动加载事件，事件对象包含 start 和 end 时间戳
dragStarted

任务条拖拽开始事件
dragMoved

任务条拖拽移动事件
dragEnded

任务条拖拽结束事件
barClick

任务条点击事件
viewChange

视图类型变化事件
expandChange

展开/收起状态变化事件
linkDragStarted

关联关系拖拽开始事件
linkDragEnded

关联关系拖拽结束事件
lineClick

关联关系线点击事件
selectedChange

任务选择变化事件
virtualScrolledIndexChange

虚拟滚动索引变化事件
#bar

任务条自定义模板（bar 类型）
#range

区间自定义模板（range 类型）
#item

任务项自定义模板
#baseline

基线自定义模板
#group

分组自定义模板
#groupHeader

分组头部自定义模板
#toolbar

工具栏自定义模板
#footer

底部自定义模板
