配置概览

ngx-gantt 提供多层次的配置方式，从全局到组件再到任务级别，满足不同场景的需求。

配置层级

  1. 全局配置：应用级别的默认配置
  2. 组件配置：特定组件的配置
  3. 任务配置：单个任务的配置

章节列表

  • 全局配置 - GANTT_GLOBAL_CONFIG Token 的结构与合并策略
  • 国际化 - 内置语言包使用与 GANTT_I18N_LOCALE_TOKEN 自定义扩展
  • 主题化 - SCSS 变量覆盖策略与运行时动态色系切换
  • 优先级原则 - 配置优先级说明
