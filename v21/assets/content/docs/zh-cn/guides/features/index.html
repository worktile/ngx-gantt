ngx-gantt 提供了丰富的功能特性，涵盖任务展示、交互操作、视图管理和性能优化等方面：

概览

  1. 任务展示：Bar 任务条、Range 任务、分组模式、Tree 模式
  2. 交互操作：拖拽移动、缩放调整、任务依赖、层级管理
  3. 视图管理：工具栏配置、视图切换、导出打印
  4. 性能优化：虚拟滚动、大数据量处理

章节列表

  • Bar 显示与交互 - 任务条拖拽、缩放、自定义模板
  • Range 任务显示 - 区间类型任务项展示、自定义模板
  • 分组模式 - 分组模式、分组展开/收起、自定义分组模板
  • 表格显示与交互 - 行拖拽排序、层级调整、列配置
  • 任务依赖 - 依赖关系创建、FS/FF/SS/SF 类型
  • Tree 模式 - 层级结构、异步加载子节点
  • 基线 - 计划与实际时间对比
  • 工具栏 - 视图切换、自定义工具栏
  • 导出打印 - 导出为图片、打印功能
  • 性能与加载状态 - 虚拟滚动、大数据量处理
