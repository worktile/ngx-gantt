功能指南

按专题拆解功能细节，解决具体的"怎么做"问题。

核心交互功能

  • Bar 交互 - 任务条拖拽、缩放、自定义模板
  • Range 区间展示 - 区间类型任务项展示、自定义模板
  • 表格交互 - 行拖拽排序、层级调整、列配置
  • 任务关联 - 依赖关系创建、FS/FF/SS/SF 类型
  • 树形与异步 - 层级结构、异步加载子节点

高级功能

  • 基准线 - 计划与实际时间对比
  • 工具栏 - 视图切换、自定义工具栏
  • 导出打印 - 导出为图片、打印功能
  • 性能优化 - 虚拟滚动、大数据量处理

学习建议

  1. 初学者：从 Bar 交互、Range 区间展示 和 表格交互 开始
  2. 进阶用户：学习 任务关联 和 树形与异步
