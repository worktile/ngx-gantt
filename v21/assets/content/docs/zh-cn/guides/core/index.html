核心概念

ngx-gantt 的核心建立在三个基础概念之上：

  1. 数据模型：如何组织和表示任务数据
  2. 视图体系：如何切换和配置视图
  3. 时间处理：如何解析、格式化和处理时间

章节列表

  • 数据模型 - GanttItem/GanttGroup 核心字段、分组模式与树形模式
  • 时间与时区 - GanttDate 时间操作、时间输入格式、时区配置
  • 视图体系 - 内置视图类型、ViewOptions 配置
