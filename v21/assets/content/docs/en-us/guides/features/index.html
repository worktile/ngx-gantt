ngx-gantt provides a rich set of features, covering task display, interaction, view management, and performance optimization.

Overview

  1. Task Display: Bar task bars, Range tasks, group mode, Tree mode
  2. Interaction Operations: drag to move, drag to resize, task dependencies, hierarchy management
  3. View Management: toolbar configuration, view switching, export & print
  4. Performance Optimization: virtual scrolling and handling large datasets

Chapter List

  • Bar Display and Interaction - drag, resize, and custom templates for task bars
  • Range Task Display - range-type task items, including custom templates
  • Group Mode - group mode, expand/collapse, and custom group templates
  • Table Display and Interaction - row drag sorting, hierarchy adjustments, and column configuration
  • Task Links - create dependency relationships and FS/FF/SS/SF types
  • Tree Mode - hierarchy structure and async loading of child nodes
  • Baseline - compare planned vs. actual time
  • Toolbar - view switching and custom toolbar
  • Export Print - export as an image and printing functionality
  • Performance and Loading Status - virtual scrolling and large data handling
