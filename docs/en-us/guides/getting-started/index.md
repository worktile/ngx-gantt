**ngx-gantt** is a modern Gantt chart component library designed specifically for Angular. It provides complete capabilities for task management, time planning, and progress tracking.

### Features

- 📋 **Task Management**: supports task creation, editing, and drag to adjust time
- ⛓️ **Dependencies**: supports dependency links between tasks (FS/FF/SS/SF) and drag to create relationships
- 📊 **Grouping & Hierarchy**: supports grouped display and infinite nested levels, and asynchronously loads child tasks
- 🎯 **Multiple Views**: supports hour/day/week/month/quarter/year views, and allows custom view types
- 🚀 **High Performance**: built-in virtual scrolling for smooth rendering of thousands of tasks
- ✨ **Rich Interactions**: bar dragging, table row dragging, selection, expand/collapse
- 🛠️ **Highly Extensible**: supports custom templates, a theming system, and custom views
- 🌍 **Multi-language Support**: includes Simplified/Traditional Chinese, English, Japanese, German, and Russian, with support for extending custom language packs
- 📏 **Baseline Comparison**: supports comparing planned vs. actual time
- 📤 **Export & Print**: supports exporting the current view as an image

### Use Cases

- 📋 **Project Management**: task planning, progress tracking, milestone management
- 🏭 **Production Scheduling**: production plans, process management, and resource scheduling
- 👥 **Team Collaboration**: task assignment, dependency management, and progress visualization

### Tech Stack Overview

ngx-gantt is built on the Angular framework. It uses `@angular/cdk` for core capabilities and relies on `date-fns` and `@date-fns/tz` to handle time and timezones.

- Angular >= 21.0.0
- @angular/cdk >= 21.0.0
- date-fns >= 4.0.0
- @date-fns/tz >= 1.0.0
