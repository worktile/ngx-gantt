ngx-gantt provides a rich set of features, covering task display, interaction, view management, and performance optimization.

## Overview

1. **Task Display**: Bar task bars, Range tasks, group mode, Tree mode
2. **Interaction Operations**: drag to move, drag to resize, task dependencies, hierarchy management
3. **View Management**: toolbar configuration, view switching, export & print
4. **Performance Optimization**: virtual scrolling and handling large datasets

## Chapter List

- [Bar Display and Interaction](guides/features/bar-interaction) - drag, resize, and custom templates for task bars
- [Range Task Display](guides/features/range) - range-type task items, including custom templates
- [Group Mode](guides/features/groups) - group mode, expand/collapse, and custom group templates
- [Table Display and Interaction](guides/features/table-interaction) - row drag sorting, hierarchy adjustments, and column configuration
- [Task Links](guides/features/task-links) - create dependency relationships and FS/FF/SS/SF types
- [Tree Mode](guides/features/tree) - hierarchy structure and async loading of child nodes
- [Baseline](guides/features/baseline) - compare planned vs. actual time
- [Toolbar](guides/features/toolbar) - view switching and custom toolbar
- [Export Print](guides/features/export-print) - export as an image and printing functionality
- [Performance and Loading Status](guides/features/performance) - virtual scrolling and large data handling
