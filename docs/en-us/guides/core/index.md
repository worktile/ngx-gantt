Understanding ngx-gantt's three core concepts (data, views, and time) is the foundation for using the component effectively.

## Basics

ngx-gantt's core is built on three fundamental concepts:

1. **Data Model**: how to structure and represent task data
2. **View System**: how to switch and configure views
3. **Time Handling**: how to parse, format, and work with time

## Chapter List

- [Data Model](guides/core-concepts/data-model) - Core fields of `GanttItem`/`GanttGroup`, Group mode and Tree mode
- [Views](guides/core-concepts/views) - Built-in view types and `ViewOptions` configuration
- [Date & Timezone](guides/core-concepts/date-timezone) - `GanttDate` operations, time input formats, and timezone configuration
- [Public Methods](guides/core-concepts/api-methods) - Guide to public methods, scrolling to a position, expand/collapse, and task queries
