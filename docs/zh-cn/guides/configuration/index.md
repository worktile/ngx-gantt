# 配置与样式

明确"配置入口"与"优先级"，实现品牌主题化和国际化。

## 配置概览

ngx-gantt 提供多层次的配置方式，从全局到组件再到任务级别，满足不同场景的需求。

### 配置层级

1. **全局配置**：应用级别的默认配置
2. **组件配置**：特定组件的配置
3. **任务配置**：单个任务的配置
