理解 ngx-gantt 的三大核心概念（数据、视图、时间）是高效使用组件的基础。

## 核心概念

ngx-gantt 的核心建立在三个基础概念之上：

1. **数据模型**：如何组织和表示任务数据
2. **视图体系**：如何切换和配置视图
3. **时间处理**：如何解析、格式化和处理时间
