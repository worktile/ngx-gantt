# 进阶定制

服务于二次封装与复杂工业级场景，提供深度定制能力。

## 章节列表

- [自定义视图](./custom-views.md) - 继承 GanttView 抽象类、覆盖 Tick 计算与格式化策略
- [高级布局](./advanced-layout.md) - 基于 ngx-gantt-root 完全拆解并重组 Side/Main 区域

## 适用场景

进阶定制功能适用于以下场景：

1. **自定义视图**：需要特殊的时间粒度或时间刻度显示
2. **高级布局**：需要完全自定义甘特图的布局结构
3. **二次封装**：基于 ngx-gantt 构建自己的甘特图组件
4. **复杂业务**：需要实现特殊的业务逻辑和交互

## 学习建议

1. **初学者**：建议先掌握基础功能和核心概念
2. **进阶用户**：学习自定义视图和高级布局
3. **高级用户**：深入源码，实现完全自定义的甘特图
