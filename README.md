A modern and powerful gantt chart component for Angular

## Installation

```bash
$ npm i @worktile/ngx-gantt --save
# or
$ yarn add @worktile/ngx-gantt
```

## Usage

```html
<ngx-gantt [start]="1514736000" end="1609430400"></ngx-gantt>
```
